# Data Structures & Algorithms (DSA)

A personal repository for practicing and studying Data Structures and Algorithms using **JavaScript**.

## Setup

This repo uses [Quokka.js](https://quokkajs.com/) for instant in-editor feedback — no need to run files manually. Just open any `.js` file in VS Code with Quokka running and results appear inline.

## Language

- JavaScript (Node.js)

## Structure

Each problem is self-contained in its own file (e.g., `problem1.js`) and includes:

- Problem description (as a comment)
- My initial solution
- Alternative solutions where applicable
